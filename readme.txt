Easy to use Wordpress plugin to accept  paypal or stripe payment for donations or membership in one click

== Description ==

Easy to use Wordpress plugin to accept paypal or stripe payment for a donation or membership or donation in one click. Can be used in posts and pages of your site.


== Usage ==

There are few ways you can use this plugin:

1. Use the sortcode [nrptt_jad] to place the payment accept form.
2. Call the function from a template file: <?php echo nrptt_jad_accept(); ?>

== Installation ==

1. Unzip and Upload the folder 'nrptt-jad' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure the options eg. your email, Subject text etc.
4. See the usage section for details on how to place the paypal payment widget
